# Machine Learning Exercises

Repositorio donde se alojan los ejercicios prácticos solicitados por el docente en **Python** para la asignatura de **Machine Learning**.

## Contenido principal

- `exercises.py` → Código donde se encuentran las funciones solicitadas por el docente en el respectivo taller.
- `tests.py` → Archivo de pruebas básicas donde se confirma el funcionamiento de las funciones desarrolladas.

## Cómo ejecutar

Simplemente ejecute el archivo tests.py y este imprimirá por consola un resumen de los datos de entrada de cada ejercicio y mostrará los datos de salida.
El único ejercicio que solicita un dato al usuario de forma directa es el primero; el resto utiliza datos de prueba quemados en el código para mayor facilidad de desarrollo.
